# BloodYatra - Blood Donor Finder Web Application

BloodLink is a web application designed to connect blood donors with those in need of blood transfusions. This platform aims to bridge the gap between blood donors and recipients, ensuring a timely and efficient supply of blood for medical emergencies.

## Features

- **User Registration and Profiles**: Users can create accounts, provide their blood type, location, and contact details, and manage their profiles.
- **Search for Donors**: Recipients can search for suitable blood donors based on blood type, location, and availability.
- **Real-time Notifications**: Donors receive notifications when there's a matching blood request in their area.
- **Emergency Alerts**: Immediate alerts can be sent out to registered donors in case of urgent requirements.
- **Blood Donation Events**: Users can stay updated on local blood donation events and drives.
- **User Ratings and Reviews**: Recipients can provide feedback and rate their experience with donors.

## Installation

1. Clone this repository: `git clone https://github.com/yourusername/bloodlink.git`
2. Navigate to the project directory: `cd bloodlink`
3. Install dependencies: `npm install`
4. Start the application: `npm start`
